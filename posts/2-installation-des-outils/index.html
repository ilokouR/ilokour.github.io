1. De quoi avons nous besoin pour apprendre Rust ?

Avant ça c’est quoi un langage de programmation ? c’est quoi programmer ?

Programmer: C’est écrire une liste de chose que l’ordinateur doit faire..

Généralement on écris les programme en utilisant les lettres alphabétiques, sauf que l’ordinateur ne comprend pas les lettres alphabétiques (A - Z), il ne comprend que : le langage binaire, un langage composé uniquement des 0 et 1.

Ecrire tout un programme avec des 0et 1, c’est compliqué, et non productif.

Par exemple: l’équivalent de A en binaire c’est: 01000001. Imaginez donc les miliers de lignes pour coder un site web.

Alors comment faire ? Et bien c’est là que les langages de programmation entrent en jeu.

Un langage de progrmmation: C’est un système de notation pour écrire des programmes compréhensibles informatiques. On utilise un ensemble d'alphabets, qui forme des mots spécifiques. cet ensemble mots sont appélés les mots clés ou réservés.

Par exemple: if, else etc… En combinants ces mots on peut donc écrire des programmes.

Lorsqu’on finis d’écrire en combinants ces mots clés, ce n’est toujours pas le binaire, on a donc besoin de traduire ça en binaire(vous savez, le seul langage que l’ordinateur comprend).

Pour traduire on utilise soit un compilateur, soit un interpréteur. Cela dépend des langages programmation, certains demandent un interpréteur, d’autres un compilateur.

  1. Un interpréteur : traduit votre fichier source ligne par ligne, à l’ordinateur. Il traduit une ligne en binaire l’ordinateur comprend, et l’exécute, en suite une autre ligne. Ansi de suite jusqu’à la fin de l’exécution du programme. Les langages interprétés (ou qui ont besoin d’un interpréteur) on a: javascript, python, php etc…

  2. Un compilateur : traduit tout votre fichier source en binanire, d’un seul coup, rassemble la traduction en un fichier qu’on appel le fichier exécutable, on donne ce dernier à l’ordinateur, qui comprend et exécute tout le programme. Les langages compilés (ou qui ont besoin d’un compilateur) on a: Rust, C, C++, Golang etc…

Le fichier source: C’est le fichier qui contient votre code écris avec les mots clés

Par analogie aux compilateur et interpréteur:

Les traducteurs des langues (à l’ONU, ou les matchs de foot) ils ont le choix:

1. soit ils font la traductions au fur et à mésure qu’on parle, c’est à dire phrase, par phrase(c’est le cas de l’interpréteur)

2. soit ils écoutent, jusqu’à la fin et après ils font la traduction d’un seul coup(c’est le cas du compilateur)

Donc si on analyse bien pour apprendre le langage Rust:

  1. Rust est compilé, donc on a besoin d’un compilateur.
  2. On a besoin d’un programme pour écrire notre code : éditeur de code.

2. Les différents choix d’installations possibles

Les plateformes en ligne :

Vous pouvez apprendre Rust sans installer quoi que ce soit sur votre machine, il existe des sites web qui mettent à votre disposition (gratuitement) : un compilateur et un éditeur de code.

Je vous recommande Rust Playground

  1. Allez sur le site
  2. Editeur de text: où écrire votre code
  3. Compilez et exécutez votre programme
  4. Observez le resultat

Les environnements de développement intégré ou IDE :

Ce sont des logiciels qui contiennent l’éditeur, le compilateur et d’autres choses utiles au développement des programmes. Ces programmes sont disponibles, sur Windows, MacOS, Linux

Je vous recommande IntelliJ Idea

Je ferais un article spécial pour la configuration de intelliJ.

Installation séparé :

Ici on installe les programmes qu’on a besoin (compilateur, éditeur de text), séparément.

C’est le choix que je vais opter pour ce cours, cela nous permettra d’apprendre à utiliser ces programmes et surtout ne pas dépendre de ces plateformes ou IDE.

  1. Sous Linux: C’est très simple

Installation du compilateur Rust

  • Allez sur Le site de Rust
  • Téléchargez Rustup (l’outil d’installation). Copiez cette ligne depuis le site

curl –proto ‘=https’ –tlsv1.2 -sSf https://sh.rustup.rs | sh

Collez sur votre terminal et appuyez sur entrez

  • Vous verez 3 options, choisissez la première
  • Après l’installation fermez puis reouvrez le terminal
  • tapez cette commande:

rustc –version

  • Vous verez la version de Rust.

  • Installation de l’éditeur de text :

Ici j’ai choisi vscode, c’est un éditeur de code riche en fonctionnalités qui nous aidera dans l’apprentissage de Rust.

Mais vous pouvez utiliser celui que vous voulez : vim, Sublime text, atome etc… Le téléchargez et lancez l’installation avec cette command dans le terminal:

sudo dpkg -i nom_du_package_que_vous_avez_.deb

  1. Sous Windows: Ce n’est pas très simple.
  • Allez sur Le site de Rust
  • Téléchargez l’éxécutable de Rustup (l’outil d’installation).
  • Lancez l’éxécutable, suivez les instructions.
  • Vous verez 3 options, choisissez la première
  • Ils vont demander d’installer Visual Studio C++ Build tools
  • Lors de l’installation de Visual Studio C++ Build tools selectionnez C++ tools.
  • Après l’installation fermez pui reouvrez le terminal
  • tapez cette commande:

rustc –version

  • Vous verez la version de Rust.

  • Installation de l’éditeur de text : Ici j’ai choisi vscode, c’est un éditeur de code riche en fonctionnalités qui nous aidera dans l’apprentissage de Rust.

Le téléchargez et lancez l’installation tout simplement

Je crois qu’on est bon, pour commencer à programmer en RUST(le langage des stars 😆)
